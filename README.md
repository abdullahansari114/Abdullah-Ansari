## 🌐 Deployment Steps

### 1. GitHub Upload
```bash
git init
git add .
git commit -m "Initial commit - SEO optimized portfolio"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
git push -u origin main
```

### 2. Netlify Deployment
1. Go to https://netlify.com
2. Click "Add new site" → "Import an existing project"
3. Select GitHub → Choose your repository
4. Deploy settings:
   - Build command: (leave empty)
   - Publish directory: `/`
5. Click "Deploy site"
6. Once deployed, go to "Domain settings"
7. Add custom domain: `abdullahansari.dev`

### 3. Google Search Console Setup
1. Go to https://search.google.com/search-console
2. Add property: `https://abdullahansari.dev`
3. Verify ownership (use DNS or HTML file method)
4. Submit sitemap: `https://abdullahansari.dev/sitemap.xml`
5. Request indexing for all pages
